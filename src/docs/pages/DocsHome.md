# Wyrd UI

A vue3 UI library based on tailwindcss, windicss and typescript.

It is inspired heavily by [Equal UI](https://github.com/quatrochan/Equal) and [Daisy UI](https://github.com/saadeghi/daisyui).

## Install

```text
yarn add wyrd-ui -D
```
